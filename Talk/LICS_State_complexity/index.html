First example

$$L_1 = \{w \in \{a,b,c\}^* : |w|_a = |w|_b = |w|_c\}$$
  • states $Q = \mathbb{Z}^2$
  • $a$ acts as $(+1,+1)$, $b$ as $(-1,0)$, $c$ as $(0,-1)$
  • initial and unique accepting state is $(0,0)$

After reading at most $n$ letters,
the automaton can be in one of $O(n^2)$ many states

State complexity


Karp (67): for an infinite automaton $\mathcal{A}$, its state complexity is $$n \mapsto \textrm{ number of states reachable by reading at most } n \textrm{ letters}$$

More examples

$$L_1 = \{w \in \{a,b,c\}^* : |w|_a = |w|_b = |w|_c\}$$ $L_1$ has quadratic deterministic state complexity
$$L_2 = \{u \sharp v \in \{a,b\}^* : u \neq v\}$$
$L_2$ has linear non-deterministic state complexity but exponential deterministic state complexity

Example: $$L_3 = \{u \sharp v \in \{0,1\}^* : u <_{\text{lex}} v\}$$
$u <_{\text{lex}} v\ $ if and only if $$\left(u_0 = 0 \wedge v_0 = 1\right)\ \vee\ \left(u_0 = v_0 \wedge u_{\ge 1} <_{\text{lex}} v_{\ge 1}\right)$$
$L_3$ has linear alternating state complexity

The results

Deterministic automata

The left quotient of $u \in A^*$ with respect to $L$ is $$u^{-1} L = \{v \mid uv \in L\}$$

Theorem (Myhill and Nerode 1958): The deterministic state complexity of $L$ is $$n \mapsto \textrm{ number of left quotients of words of length } n$$

Alternating automata


Theorem: For $n \in \mathbb{N}$, the alternating state complexity of $L$
is lower bounded
by the number of generators of the lattice containing all left quotients of words of length $n$

Prime numbers

$$\text{Prime} = \{u \in \{0,1\}^* : \text{bin}(u) \text{ is prime}\}$$

Theorem (Hartmanis and Shank 1969): Prime does not have subexponential deterministic state complexity


Relies on Dirichlet's theorem on primes in arithmetic progressions

New results


Theorem: Prime does not have sublinear alternating state complexity


Relies on a deep result of Maier and Pomerance (1990) about sparsity of prime numbers

Theorem: Assuming Dickson's conjecture, Prime does not have subexponential alternating state complexity
