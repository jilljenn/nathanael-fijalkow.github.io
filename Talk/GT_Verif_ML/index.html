Safety Verification of Neural Networks

Model-checking problem

INPUT: a (trained) neural net representing $f : \mathbb{R}^d \to \mathbb{R}^d$ and a specification $$\phi(x) \implies \psi(f(x))$$ OUTPUT: does $f$ meet the specification?

Theorem: NP-complete

Finding adversarial examples

Approach

  • Define a set of manipulations on images
  • Given $x$, does there exist a safe region around $x$?
  • Construct regions inductively in a backward fashion walking up the layers
  • Computational tasks expressed in a SMT solver

The simplex approach

Based on the CAV 2017 paper

Reluplex: An Efficient SMT Solver for Verifying Deep Neural Networks


Guy Katz, Clark Barrett, David Dill, Kyle Julian and Mykel Kochenderfer

Objective

Model-checking specifications $$\phi(x) \implies \psi(f(x))$$ with $\phi,\psi$ boolean combinations of linear inequalities

Approach

  • Express in SMT with explicit ReLU
  • Simplex-based algorithm with lazy evaluation for ReLU
  • Floating point arithmetic for performances
