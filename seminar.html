Speaker: Alexander Clark, King's College

Title: Exact query learning of automata and context-free grammars.

Abstract: In this talk we will discuss learning automata and context-free grammars in the exact query framework (Angluin, 1987). We will start by discussing the general problem of learning when the instance space consists of discrete combinatorial objects like strings (trees, graphs and so on), and the minimally adequate teacher learning model, where the learner only has access to two sorts of queries: membership queries and equivalence queries. We will then present Angluin's L-star algorithm for learning all regular sets via deterministic finite state automata, and extensions of this approach to the problem of learning interesting subclasses of context-free grammars, and mildly context-sensitive extensions thereof.
